# Meet the Bandwagons — World Cup 2026 Draw

A shared live draw for up to 16 friends. Everyone pulls 3 teams from a hat of all 48 World Cup 2026 nations. One draw per person, live board for all.

## How it works

- First visitor creates a JSONBin and the URL updates to `yoursite.com/#abc123`
- Share that full URL — everyone with the same `#` code shares one hat
- 48 teams, 3 per person, max 16 players
- No accounts or logins required

## Run locally

```bash
npm run dev
```

Open http://localhost:3000 (or whatever port `serve` picks).

## Deploy

Drag the folder to [Netlify Drop](https://app.netlify.com/drop), push to GitHub Pages, or deploy anywhere that serves static files. No build step, no env vars.

## Reset

**Reset everything** wipes all draws for everyone on that shared bin — use only to start over.
